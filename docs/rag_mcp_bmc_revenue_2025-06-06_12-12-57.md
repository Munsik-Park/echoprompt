# ✅ 수익 모델 (Revenue Model)

## 1. 구독형 SaaS
| 플랜            | 가격 (예시)  | 기능 |
|:----------------|:------------|:-----|
| **Free**        | $0          | 기본 대화 모드, 세션 저장 제한 |
| **Pro**         | $15/월       | 긴 맥락 유지, Git 연동, 1M 토큰 포함 |
| **Team**        | $99/월/팀당 5명 | 팀 협업, 대화 공유, 긴 세션 지원 |
| **Enterprise**  | Custom      | 대규모 벡터 검색, API 확장, 보안 강화 |

## 2. 사용량 기반 과금 (Usage-based)
- 추가 토큰 사용: 100K 토큰당 $0.10
- 추가 저장소 사용 (벡터DB): 1GB당 $2/월

## 3. 부가 수익 모델
- 커스텀 모델 Fine-tuning (기업 대상)
- On-premise 설치형 라이선스
- GitHub Apps/Plugins 수익화 (Marketplace 배포)
